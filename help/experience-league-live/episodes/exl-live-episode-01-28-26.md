# Agenter på jobbet: Skapa och aktivera målgrupper och resor

Modern marknadsföring går snabbt framåt. På det här webbseminariet visar Adobe produktteam hur Adobe Experience Platform kan ge dynamisk artificiell intelligens i vardagliga arbetsflöden, så att marknadsförarna kan identifiera, bygga och aktivera snabbare med tillförsikt.

Se hur Audience Agent hittar målgruppsinsikter och skapar färdiga målgrupper på några minuter. Se sedan Journey Agent förvandla språkinställningar till flerstegsresor från målgruppskompetens till flerkanalsaktivering.  Ni kommer att se hur agenter inom Real-Time CDP och Journey Optimizer minskar friktionen för att leverera kundupplevelser i stor skala för både affärsanvändare och tekniska användare.
