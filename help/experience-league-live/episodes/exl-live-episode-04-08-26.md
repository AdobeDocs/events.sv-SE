# Nya användningsexempel för kundengagemang: iOS Live Activity, Web Push och Persistent Messaging

Utforska nya ytor och användningsexempel för kundengagemang när varumärken rör sig bortom enkelkanalsmeddelanden. Upptäck hur marknadsförings- och produktteam kan skapa mer synliga, kontextuella och permanenta upplevelser som driver ett kontinuerligt engagemang och mer omfattande realtidsinteraktioner över olika kanaler.

>[!VIDEO](https://video.tv.adobe.com/v/3483389/?quality=12&learn=on)

Fortsätt diskutera programmet i [Experience League Community Post.](https://experienceleaguecommunities.adobe.com/adobe-journey-optimizer-15/experience-league-live-new-use-cases-for-customer-engagement-ios-live-activities-web-push-and-persistent-messaging-249581){target="_blank"}

<br>
</br>
I programmet utforskade vi hur organisationer som använder Adobe Journey Optimizer kan utöka sina kundinteraktionsstrategier med iOS Live Activity, Web Push Notifications och Message Inbox. Tillsammans gör dessa nya kanaler och funktioner det möjligt för team att leverera uppdateringar i realtid, utöka engagemanget bortom mobilappen och se till att viktiga meddelanden alltid är tillgängliga - och skapa mer enhetliga, kundcentrerade resor.

Vi täckte:

- iOS Live-aktiviteter: Leverera uppdateringar i realtid på iPhone Lock Screen och Dynamic Island för värdefulla stunder som orderspårning, flygstatus och livepoängen - så att kunderna hela tiden informeras om händelserna.

- Push-meddelanden för webben: Öka engagemanget bortom mobilappar genom att nå kunder direkt i webbläsaren, direkt engagera besökare även när webbplatser inte är aktivt öppna med samma användararbetsflöden som mobilpush i Journey Optimizer

- Inkorgen för meddelanden: Skapa en beständig, varumärkesprofilerad inkorg där kunderna kan besöka viktiga meddelanden, skräddarsytt innehåll och erbjudanden på nytt, och minska antalet missade meddelanden samtidigt som de förbättrar relevansen och användarupplevelsen.
