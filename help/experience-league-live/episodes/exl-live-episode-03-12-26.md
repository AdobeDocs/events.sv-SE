# Lär känna kampanjsamordning i Adobe Journey Optimizer

Se hur kampanjsamordning i Journey Optimizer hanterar era stora, en-till-många-program för flerkanalsmarknadsföring - från målgruppsframtagning till lansering.

Vi täckte kampanjens arbetsyta och relationsscheman som grund för hantering av komplexa kampanjer och kundrelationer, segmentering av flera enheter för att rikta sig till rätt målgrupper över komplexa data, målgruppsförbättring on-demand och antal före sändning för att vara korrekta, och hur detta sömlöst fungerar i Journey Optimizer idag.
