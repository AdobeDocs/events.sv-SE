# 2026.03.0 AEM Release Overview Video

>[!VIDEO](https://video.tv.adobe.com/v/3483060/?learn=on&enablevpops)

Funktionerna i den här versionen fokuserar på innovationer i Experience Manager Sites, Forms, Assets och Cloud Foundation och inkluderar följande:

* Webbplatser
   * Experience Production Agent - uppdateringar och startfunktioner

* Forms
   * Stöd för arbetsflöden i IC Editor
   * Stöd för Theme Editor
   * Versionshantering för Forms

* Resurser
   * AEM Content Advisor - GA med Adobe-appar
   * Microsoft Office-anslutning (PowerPoint, Word, Excel)

* Cloud Foundation
   * Styrningsagent: Varumärkeshantering
   * Granska behörigheter för pekrörelser
   * OIDC-utloggningsfunktion
   * Pausa underhållsuppdateringar

* Cloud Manager
   * Cloud Manager MCP Server
