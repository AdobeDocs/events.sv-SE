# Migrering till AEM Cloud Service

Under den här sessionen kommer du att få en presentation av de tekniker och verktyg som finns tillgängliga för att underlätta migreringen till AEM Cloud-tjänsten. Från innehållsöverföringsverktyget till nya Cloud Acceleration Manager som används tillsammans med Best Practice Analyzer för att utvärdera din miljö, missa inte den här aktiveringssessionen för att vara säker på att du är redo för ett framgångsrikt publiceringstillfälle på Cloud Service.
