# AEM Champion Session Multi Tenancy

Sessionen fokuserar på hur flera klientorganisationer kan finnas på samma AEM as a Cloud Service-instans. Fokus ligger på att använda Adobe Cloud Manager för att driftsätta flera koddatabaser i samma AEM-instans. Detta inkluderar konfiguration av olika Git-databaser samt användning av Git-undermoduler.
