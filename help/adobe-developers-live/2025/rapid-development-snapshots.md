# Freeze, Restore, Innovate: Hands-On with Rapid Development Environment Snapshots for Cloud Development

Upptäck hur Rapid Development Environment (RDE) i Adobe Experience Manager Cloud Service möjliggör snabba, molnbaserade iterationer. Lär dig hur utvecklare kan frysa, återställa och återanvända fullständiga RDE-tillstånd med hjälp av snapshot-funktioner. Detta ger er möjlighet att effektivisera testningen, återställa omedelbart och snabba upp utvecklingscyklerna med Adobe I/O CLI RDE plugin. Förbättra effektiviteten i molnutvecklingen och anpassa er efter företagets styrningspolicyer.

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

Denna session - Freeze, Restore, Innovate: Hands-On with Rapid Development Environment Snapshots for Cloud Development - innehåller Remo Liechti, Software Development Engineer på Adobe, och spelades in i förväg för Adobe Developers Live 2025. Remo visar hur Rapid Development Environment (RDE) möjliggör snabb, molnbaserad upprepning i Adobe Experience Manager Cloud Service och hur nya funktioner för ögonblicksbilder gör att utvecklare kan frysa, återställa och återanvända fullständiga RDE-tillstånd. Lär dig hur du effektiviserar testning, direkt återställning och snabbar upp utvecklingscyklerna i AEM Cloud med Adobe I/O CLI RDE-plugin.

Ett särskilt tack till våra sponsorer Algolia och Ensemble för att de stöder Adobe Developers Live 2025.
