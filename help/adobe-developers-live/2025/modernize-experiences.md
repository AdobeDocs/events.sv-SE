# Migrera och moderera upplevelser med Adobe Experience Manager for User Engagement och AI-native Discovery

Se hur Adobe Experience Manager omvandlar företagswebbplatser för AI-eran. Se hur AI-agenter automatiserar webbplatsmigreringen, förbättrar innehållet och förbättrar upptäckbarheten, vilket minskar tiden för start. Få insikter i hur ni integrerar konversationsgränssnitt, skapar nya sidor och förbättrar designen för bättre användarengagemang.

>[!VIDEO](https://video.tv.adobe.com/v/3477717/?learn=on&enablevpops)

Detta seminarium - Migrera och moderera upplevelser med Adobe Experience Manager for User Engagement och AI-native Discovery - innehåller Giorgia Fiscaletti, Cloud Reliable Engineer på Adobe och Paulo Mottadelli, Director of Engineering på Adobe. Den här presentationen spelas in live från Basel och visar hur Adobe Experience Manager Agents moderniserar företagswebbplatser för AI-eran. Lär dig hur AI-agenter automatiskt migrerar, berikar och standardiserar innehåll för generativ identifiering, hantering av blockimplementering och generering av design, vilket minskar tiden för start och förbättrar innehållsupptäckt.

Ett särskilt tack till våra sponsorer Algolia och Ensemble för att de stöder Adobe Developers Live 2025.
