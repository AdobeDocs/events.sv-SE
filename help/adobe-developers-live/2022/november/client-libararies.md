# Forms som fungerar - Utnyttja klientbibliotek och Headless Forms

Adobe Experience Manager Forms har släppt många spännande nya funktioner på Cloud Service. För formulärutvecklare finns det ibland behov av att utnyttja anpassade skript och JavaScript-bibliotek från tredje part för att förbättra Adobe Experience Manager Forms användarupplevelse.

Under den här sessionen ska vi visa hur du använder klientbibliotek och tar ut skriptfunktionerna från formulärförfattare så att de enkelt kan välja funktioner som behövs för att utföra avancerade beräkningar, datumhantering och anpassade modala fönster.

Vi kommer också att visa upp våra senaste innovationer med Headless Adaptive Forms, där du kan skapa ett formulär en gång och använda det i flera kanaler, bland annat via webben, SPA-appen med hjälp av reaktion och anpassa den responsiva appen.

Våra produktexperter visar hur man använder samma formulär för datainhämtning i interaktiva kanaler som WhatsApp, så att ni kan få formulär att fungera exakt som ni vill.
