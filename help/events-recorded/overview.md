# Adobe Experience League - inspelningar {#events}

## Med Experience League är lärande en händelse.

Håll dig uppdaterad om exklusiva event och webbinarier för att utöka din inlärning. Besök den här platsen ofta för att se inspelningar av tidigare händelser som Experience League LIVE, SDödbyggare, Developers Live med mera. Om du vill registrera dig för ett kommande evenemang kan du titta närmare på [kommande evenemang](https://%65xperienceleague.adobe.com/events/?lang=en) på Experience League.
