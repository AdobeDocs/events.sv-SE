# Skalbar aktivering - Förvandla Workfront-utkast till Real-World Training-program

I takt med att organisationen växer blir det också mycket komplicerat att komma igång, utbilda och stödja användare. Det som fungerar för ett litet pilotteam delas ofta upp i stor skala. Där kan Adobe Workfront Blueprints och Project Templates ge struktur, konsekvens och snabbhet. Under det här seminariet tittar vi på hur Workfront kan driva motorn för utbildning och driftsättning, som drivs av utkast.
