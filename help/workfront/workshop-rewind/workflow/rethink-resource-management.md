# Ett nytt sätt att se på resurshantering på det flexibla sättet hos rådgivare Excel

Se hur det kreativa teamet på Advisors Excel upptäckte ett praktiskt och smidigt sätt att hantera resurser - med köer, nivåer och prioritetsbedömning, helt inifrån Workfront, genom sin treåriga omvandling från en manuell, strikt resurshantering till ett köbaserat system i Workfront. Följ Lindsey Brown, Creative Operations Product Manager och Cathy Clifton, Project Coordinator Director, och förklara hur man kan flytta arbetsuppgifterna till delade köer, införa strukturerade eftersläpningar och omdefiniera teamroller, vilket ger ökad flexibilitet, skalbarhet och synlighet samtidigt som man minskar stress och flaskhalsar.
