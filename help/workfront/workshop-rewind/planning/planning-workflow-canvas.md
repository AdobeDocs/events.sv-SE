# Workfront Power Trio - Ansluta planerings-, arbetsflödes- och arbetsytekontrollpaneler

Framgångsrika organisationer behandlar inte planering, utförande och rapportering som separata aktiviteter, de kopplar ihop dem. Under den här sessionen visar Workfront Expert, Madalyn Destafney, hur man kan utnyttja Workfront Planning tillsammans med Workfront Workflow för att skapa en ansluten operativmodell som smidigt går från planering till körning, med synlighet i realtid via Canvas Dashboards.
