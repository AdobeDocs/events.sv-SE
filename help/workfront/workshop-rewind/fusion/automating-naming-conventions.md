# 5 tips för att automatisera namnkonventioner med Workfront Fusion

Att hålla ordning på Workfront-miljön börjar med enhetliga namnkonventioner, men det kan vara tidsödande och fel lätt att införa dem manuellt. På det här webbinariet som ligger i samarbete med Adobe Professional Services och Field Engineering kan du lära dig hur du använder Workfront Fusion för att automatiskt tillämpa namngivningsstandarder för projekt, uppgifter, problem med mera.

Vi kommer att gå igenom exempel från verkligheten om hur man använder Fusion för att:

- Generera projektnamn dynamiskt baserat på formulärindata eller metadata
- Standardisera uppgifts- och problemnamnen för enkel rapportering och synlighet
- Använd ökande siffror för att etikettera objekt och se till att instansen är unik

Oavsett om du just har börjat med Fusion eller vill utöka dina automatiseringar kan du ge dig av med praktiska exempel och inspiration för att effektivisera konfigurationen av Workfront.
