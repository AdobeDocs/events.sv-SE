# Från data till design - automatisera profilerade PDF:er med Workfront Fusion

Förvandla Workfront-data till prydliga, färdiga produkter för intressenter!  I samarbete med Brian Haight och Pete Worth på Adobe Professional Services fokuserade mötet på hur man kan använda Workfront Fusion för att hämta in data och automatiskt generera varumärkesanpassade, mallsidiga PDF:er som kan bifogas direkt till Workfront-objekt och delas med intressenter - utan manuell inklistring. De gick igenom användningsexempel och beprövade metoder i verkligheten, så att ni kan börja automatisera med tillförsikt.
